# CommentRemover
(🚧开发中)  
(⚠谨慎使用)

源代码注释删除工具，一键删除各种源代码内的注释  
使用之前请务必备份源文件，以避免程序出现可能的bug🐞，在确认后再做操作。数据无价！

## 支持类型
|文件类型|注释格式|
|---|---|
|`.c.cpp` `.java` `.js` `.swift` `.rs` `.php`|`// Comment` `/* Multi-line comment */`|
|`.py` `.rb` (🚧)|`# Comment`|
|`.go` (🚧)|`// Comment`|
|`.html` (🚧)|`<!-- Comment -->`|
|`.css` (🚧)|`/* Comment */`|
